Test your URL Pattern against URLs

What is this?

This is a service that enables you to find the matches of your URL Pattern against URLs or individual URL components. The URL Pattern API is used by the URLPattern interface inside of Portal.

How does it work?

We use the exec() method of the URL Pattern interface and display either a match or miss. Find out more and deploy your own URL Pattern Matcher on Deno by visiting GitHub.
